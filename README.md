# Healthcare-Institutions-SQL-Analysis-ODHF-Project-
This project explores the Open Database of Healthcare Facilities (ODHF) to understand how healthcare institutions are distributed across Canada. It includes database design, data cleaning, SQL queries, subqueries, CTEs, and reusable views. The work demonstrates how SQL can transform raw public data into meaningful insights.
